# Missile Evasion Simulation

6-DOF simulation of an F-16 Fighting Falcon executing a defensive break turn to evade an AIM-9 Sidewinder. Built on the [stellarust](https://github.com/navierai/stellarust) simulation framework.

## Scenario

- **Target vehicle:** F-16 Fighting Falcon at 5,000 m altitude, Mach ~0.9
- **Threat:** AIM-9 Sidewinder-class IR missile, launched 3 km behind and 500 m offset
- **Evasion maneuver:** Hard right break turn at 80° bank, ~7.5 G sustained — maximizes angular crossing rate and forces the missile to exceed its lateral acceleration limit

The simulation runs both vehicles simultaneously at 1 kHz physics / 100 Hz flight software, computes miss distance at closest point of approach, and writes a `.stella` bundle for 3D visualization.

## Architecture

```
src/
  main.rs          # Scenario setup, sim harness, results analysis, output
  vehicles.rs      # Vehicle parameters (mass, inertia, thrust, initial states)
  f16_fsw.rs       # F-16 flight software: aerodynamic force model + break-turn autopilot
  missile_fsw.rs   # AIM-9 flight software: augmented proportional navigation guidance
gen_charts.py      # Post-processing: generate engagement geometry and telemetry charts
```

### F-16 Flight Software

Three-phase autopilot:
1. **Cruise** — level flight, engine at 40% thrust until radar contact
2. **Break Turn** — immediate hard right bank (80°), max AoA, 95% thrust for 12 seconds
3. **Extend** — wings level, accelerate away

Aerodynamic force model uses a linear lift curve (C_Lα = 3.44 /rad) with parabolic drag polar, exponential atmosphere, and 9 G structural limit.

### Missile Flight Software

Augmented proportional navigation (N = 4) with two phases:
1. **Boost** — full thrust with thrust-vector control steered toward PN guidance command
2. **Terminal** — coast/partial thrust, aerodynamic fin steering

Max lateral acceleration: 30 G.

## Results

The simulation prints miss distance and F-16 flight diagnostics to stdout and writes:

- `results/evasion.stella` — 3D scene bundle (open with [Stella](https://stella.navier.ai))
- `results/charts/` — PNG charts generated by `gen_charts.py`

Charts include:
- `engagement_geometry.png` — top-down trajectory plot with closest point of approach
- `range_vs_time.png` — missile-to-F-16 range over time
- `f16_flight_params.png` — bank angle, G-load, altitude, speed
- `missile_guidance_params.png` — closing speed, lateral acceleration, fuel
- `mode_timeline.png` — phase timeline for both vehicles

## Prerequisites

**Rust toolchain:** Install via [rustup](https://rustup.rs)

**stellarust registry access:** The `stellarust-*` crates are hosted on the Navier AI private Cloudsmith registry. Set your credentials:

```bash
cargo login --registry navierai <your-token>
```

Contact [cameron@navier.ai](mailto:cameron@navier.ai) to request registry access.

**Python (for charts only):**

```bash
pip install numpy pyarrow matplotlib
```

## Running

```bash
# Run the simulation
cargo run --release

# Generate analysis charts (after simulation completes)
python gen_charts.py
```
